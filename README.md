# 📰 Fake News Detection System

## 📌 Project Overview
This project is a Machine Learning based Fake News Detection System that analyzes news text and predicts whether the news is **Real or Fake**. It helps in identifying misleading or false information using Natural Language Processing (NLP) techniques.

## 🎯 Objective
To build an AI model that can automatically classify news articles as:
- ✔ Real News  
- ❌ Fake News  


## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing (NLP)
